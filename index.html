vue-add-to-calendar

A Vue.js component that provides "Add to Calendar" functionality

Demo:

  • Add to Google calendar
  • Add to Microsoft live calendar

Installation

{% highlight bash %} # Yarn yarn add vue-add-to-calendar # NPM npm install --save vue-add-to-calendar # Bower bower install vue-add-to-calendar {% endhighlight %}

Usage

Loading the library

Browserify / Webpack
{% highlight javascript %} var AddToCalendar = require('vue-add-to-calendar'); Vue.use(AddToCalendar); {% endhighlight %}
